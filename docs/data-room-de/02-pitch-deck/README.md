# AlltagsEngel Premium Investor Pitch Deck v2

## Überblick

Professionelles Investor Pitch Deck für AlltagsEngel UG mit 14 Folien, vollständig im Dark-Mode Design Language der App gestaltet.

## Design-Spezifikationen

### Farben (exakt vom App-Design)
- **Coal (Primary BG):** 1A1612
- **Coal2:** 252118
- **Coal3:** 332E24
- **Coal4:** 443C2E
- **Gold (Primary Accent):** C9963C
- **Gold2 (Lighter):** DBA84A
- **Ink (Light Text):** F7F2EA
- **Ink2:** C4B8A8
- **Ink3:** 9A8C7C
- **Ink4:** 7A6E5E
- **Green (Success):** 5CB882
- **Red (Alert):** D04B3B

### Typografie
- **Headlines:** Palatino Linotype (28-48pt)
- **Body Text:** Calibri Light (11-14pt)
- **Accents:** Calibri Bold
- **Section Labels:** UPPERCASE, charSpacing 4-6pt

### Gestaltungselemente
- Border-Radius: 18px (0.15 in PowerPoint)
- Card Backgrounds: Coal2 oder Coal3
- Subtle Borders: Coal4, 1pt
- Shadows: outer, 6pt blur, 135° angle, 15% opacity
- Separator Lines: Gold, 0.4" wide, 2px height
